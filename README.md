AI Budget Planner
일반 가계부에, AI를 활용으로 소비 패턴을 분석하고 맞춤형 예산 가이드라인을 제안하는 웹 애플리케이션.

--Tech Stack--
[Backend]
Python 3.14.3 / FastAPI: 비동기 API 서버 구축
Gemini API: gemini-2.5-flash
Database Client : Supabase(PostgreSQL)

[Frontend]
React: 컴포넌트 기반 UI 구현
Tailwind CSS: 스타일링 및 반응형 디자인
Vercel (Deployment)

Key Features
Budget Analysis: 고정 및 변동 지출 데이터를 기반으로 한 예산 분석
AI Report: 소비 습관에 따른 저축 전략 조언 생성
Responsive Dashboard: 모바일 최적화 화면 제공
