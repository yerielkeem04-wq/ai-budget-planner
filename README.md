AI Budget Planner
AI를 활용해 소비 패턴을 분석하고 맞춤형 예산 가이드라인을 제안하는 웹 애플리케이션입니다.

Tech Stack

Backend
Python / FastAPI: 비동기 API 서버 구축
OpenAI API: 사용자 맞춤형 소비 전략 리포트 생성
Pydantic: 데이터 검증 및 설정 관리

Frontend
React: 컴포넌트 기반 UI 구현
Tailwind CSS: 스타일링 및 반응형 디자인

Key Features
Budget Analysis: 고정 및 변동 지출 데이터를 기반으로 한 예산 분석
AI Report: 소비 습관에 따른 저축 전략 조언 생성
Responsive Dashboard: 데스크톱 및 모바일 최적화 화면 제공
